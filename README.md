# Heart Disease Risk Analysis

## 🎯 Objective
This project analyzes medical data to identify key clinical parameters, demographic trends, and risk factors associated with heart disease.

## 🛠 Tools Used
* **Python**
* **Libraries:** Pandas, Matplotlib, Seaborn
* **Techniques:** Clinical Data Visualization, Pattern Recognition

## 📈 Key Insights
* **Demographics:** The majority of patients in the dataset fall within the 40–60 age range.
* **Clinical Indicators:** A positive correlation exists between higher cholesterol levels and heart disease diagnosis.
* **Prevalence:** Approximately 33% of patients report exercise-induced angina, and chest pain types 2 and 3 are more frequent in diagnosed patients.
* **Gender Factors:** The analysis highlights differences in clinical presentation across the dataset.

## 🚀 Conclusion
By identifying high-risk parameters, this analysis supports a better understanding of the clinical factors that correlate with heart health.
